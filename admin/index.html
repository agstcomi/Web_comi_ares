Configurar Base de Dades Supabase

Per a sincronitzar les teues dades al núvol de manera gratuïta i allotjar la web en producció, crea un projecte en Supabase, configura les taules corresponents i introdueix les credencials a continuació.

Còpies de Seguretat i Sincronització

Descarrega una còpia de seguretat de totes les teves notícies, actes i galeria local, o restaura dades prèviament exportades des d'un arxiu JSON. Això et permetrà passar fàcilment les dades del mode local a la teva base de dades en línia de Supabase.

ℹ️ Estructura SQL de les Taules en Supabase

Si uses Supabase, has de crear aquestes 3 taules en el teu projecte (Editor SQL):

-- Taula de Notícies (news)
CREATE TABLE news (
  id TEXT PRIMARY KEY,
  created_at DATE DEFAULT now(),
  updated_at DATE,
  title TEXT NOT NULL,
  subtitle TEXT,
  content TEXT NOT NULL,
  image_url TEXT,
  author TEXT DEFAULT 'Comissió de Festes',
  additional_images TEXT[] DEFAULT '{}'::text[]
);

-- Taula d'Esdeveniments (events)
CREATE TABLE events (
  id TEXT PRIMARY KEY,
  title TEXT NOT NULL,
  description TEXT NOT NULL,
  date DATE NOT NULL,
  time TEXT NOT NULL,
  location TEXT NOT NULL,
  category TEXT NOT NULL,
  long_description TEXT,
  image_url TEXT
);

-- Taula de Galeria (photos)
CREATE TABLE photos (
  id TEXT PRIMARY KEY,
  created_at DATE DEFAULT now(),
  title TEXT NOT NULL,
  image_url TEXT NOT NULL,
  category TEXT NOT NULL
);
